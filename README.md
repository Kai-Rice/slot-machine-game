# slot-machine-game
A simple slot machine game created using Python that runs in a terminal.

#### Features
- Functional slots and unique prize values.
- Dynamic menu screens and player statistics.
- In-game store that allows the player to purchase new slot icons.
- Simple error handling.

#### Commamds
```
store - Open the store to buy slot packs 
reels - Set number of reels
stats - View player stats
restart - Restart the game
add - Add money to wallet
wallet - View wallet balance
quit - Quit the game         
help - View this help message
```
